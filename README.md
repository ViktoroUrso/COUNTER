Идея специализированного приложения для визуализированного подсчёта трафика

Цель разработки программы - отработать знания, умения и навыки разработки мобильных приложений для ОС андроид на языке Python.

Программа для android 
Целевая аудитория программы: организаторы и исполнители полевых маркетинговых исследований по подсчету трафика в конкретной точке.


Возможности:
- автоматическая фиксация времени проведения подсчета (начало - окончание) - в отчете по подсчёту трафика
- автоматическая фиксация места проведения подсчёта - в отчёте по подсчёту трафика
- отдельные кнопки с картинками для элементов стандартной поло-возрастной структуры трафика (дед-бабка; мужик-тетка; парень-девушка; мальчик-девочка; малыш - малышка)
+ автоматический итоговый отчёт по завершению подсчёта трафика, который может быть отправлен (расшарен) через разные каналы в формате csv, xls, docx
